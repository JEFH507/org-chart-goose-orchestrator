## Archive Policy

**Files archived when**:
1. Superseded by newer comprehensive documentation
2. Session-specific artifacts (resume reports, handoffs)
3. Historical test results from completed phases
4. Planning documents from completed phases

**Files NOT archived**:
- Active progress logs (phase0-5-progress.md)
- Active test results (phase5-test-results.md)
- ADRs (architectural decisions are permanent)
- User guides (evergreen content)
