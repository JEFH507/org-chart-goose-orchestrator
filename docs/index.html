CE vs SaaS (at a glance)

  • Identity: CE Keycloak (OIDC) • SaaS any OIDC (Okta/Azure/Auth0)
  • Secrets: CE Vault OSS + Transit • SaaS managed Vault + cloud KMS
  • Storage: CE Postgres + MinIO • SaaS managed SQL + S3/GCS
  • Guard: Agent pre/post (local) • Provider wrapper ON by default in SaaS
  • Messaging: HTTP‑only in MVP • Optional bus adapters later
  • Telemetry: Structured logs by default • OTel optional
